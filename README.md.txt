# Telegram Moderation Bot

Бот для модерации чатов с системой голосований и рейтинга.

## Функции
- Система жалоб с сохранением контента
- Голосования за бан/мут
- Рейтинг участников по реакциям
- Панель модерации для администраторов

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Настроить переменные окружения
4. Запустить: `python src/bot.py`

## Команды
- `/start` - Запустить бота
- `/report` - Пожаловаться на сообщение
- `/vote_ban` - Голосование за бан
- `/rating` - Рейтинг участников
- `/moderate` - Панель модерации
